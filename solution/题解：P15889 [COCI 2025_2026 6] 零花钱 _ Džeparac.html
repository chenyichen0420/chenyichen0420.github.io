思路分析

本质上就是在询问不同的和不超过 n 的偶数数列个数。

于是我们将 2n​→n，就变成了询问和不超过 n 的正整数数列的个数。

我们考虑 dpi​ 表示和恰好为 i 的正整数序列的方案数。其中 dp0​=1。

于是显然 dpi​=1+∑j=0i−1​dpj​。

断言 dpi​=2i，下证：

i=0 时显然成立，假定 i<i0​ 时均成立，那么有 dpi0​​=1+∑j=0i0​−1​dpj​=1+(2i0​−1)=2i0​，于是 i≤i0​ 时均成立，证毕。

代码如下：

#include<iostream>
using namespace std;
long long n;
inline int qpow(long long a, long long b, long long mod) {
	int ret = 1;
	for (;b;b >>= 1, a = a * a % mod)
		(b & 1) && (ret = ret * a % mod);
	return ret;
}
signed main() {
	ios::sync_with_stdio(0); cin >> n; n /= 2;
	cout << qpow(2, n, 1000000007) << endl;
}
