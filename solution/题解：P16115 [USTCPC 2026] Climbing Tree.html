思路分析

好题，下面提供一种思路较为自然的解法。

我们考虑：如果一个点在同一层兄弟节点间进行过左移动和右移动，那么他此时就可以处在这一层的任何一个位置。

于是，我们对于每一个点，记录一下他的相对深度，以及他可以在这一层的哪些相对位置。

欸，这个相对位置我们怎么描述啊？一个很合理的想法是直接指定任意一条穿过原节点的直上直下的链为基准，如果主链向左，定义为左节点；如果主链向右，定义为右节点。

我们维护他与主链的相对位置。为了辅助维护，我们还需要记录他在不在主链毛毛虫上。如果不在，那么显然不管怎么移动，和主链的相对位置都是锁死的，不会变，反之则可能任意变化。

于是呢，状态可以用三个部分表示出来：

  1. 深度
  2. 与主链的相对位置：包括主链节点，左节点，右节点，任意节点
  3. 是否锁定状态

此题可解。考场代码如下：

#include<iostream>
#include<string>
#include<algorithm>
#include<numeric>
#include<vector>
#include<set>
#include<cmath>
#include<queue>
#include<unordered_map>
#include<cassert>
#include<map>
#include<cstring>
#include<stack>
using namespace std;
#define int long long
int t, n, sz; char c[100005];
struct status {
	int dep, pos, mod;
}s[100005]; stack<status>dst;
/*
我们指定从当前点向上的链，向下任意链为主链
如果主链向左，定义为左节点；
如果主链向右，定义为右节点；
状态可以用三个部分表示出来：
1. 深度
2. 与主链的相对位置
	包括主链节点，左节点，右节点，任意节点。
3. 是否锁定状态
*/
signed main() {
	ios::sync_with_stdio(0);
	for (cin >> t;t;t--) {
		cin >> (c + 1); n = strlen(c + 1);
		s[sz = 1] = { 0,0,0 };
		for (int i = 1;i <= n;++i) {
			if (c[i] == 'p') {
				//判断有问题
				if (dst.size()) s[sz + 1] = dst.top(), dst.pop(), sz++;
				else s[sz + 1] = { s[sz].dep - 1,0,0 }, sz++;
			}
			else if (c[i] == 'c') {
				dst.emplace(s[sz]);
				if (s[sz].mod || s[sz].pos == 2 || s[sz].pos == 1)
					s[sz + 1] = { s[sz].dep + 1,s[sz].pos,1 }, sz++;
				else s[sz + 1] = { s[sz].dep + 1,3,0 }, sz++;
			}
			else if (c[i] == 'l') {
				if (s[sz].mod) s[sz + 1] = { s[sz].dep,s[sz].pos,1 }, sz++;
				else s[sz + 1] = { s[sz].dep,s[sz].pos | 1,0 }, sz++;
			}
			else {
				if (s[sz].mod) s[sz + 1] = { s[sz].dep,s[sz].pos,1 }, sz++;
				else s[sz + 1] = { s[sz].dep,s[sz].pos | 2,0 }, sz++;
			}
		}
		if (s[sz].dep == 0 && (s[sz].pos == 0 || s[sz].pos == 3) && !s[sz].mod) cout << "Yes\n";
		else cout << "No\n";
		while (dst.size()) dst.pop();
	}
}
