思路分析

因为绝对值和随意交换，ai​ 和 bi​ 是等价的。我们限制 ai​≤bi​。

首先，不难想到，你不管怎么选，对手都能够让这个结果变的不劣，分为两种情况：

  1. 两个“区间”不相交，也就是原先的满足 ai​≤bi​<aj​≤bj​。那么对手交换顺序，可以做到贡献为 (bj​−ai​)+(bi​−aj​)。比初始状态下的 (bi​−ai​)+(bj​−aj​) 多了 2(aj​−bi​)。
  2. 两个“区间”相交，也就是原先的满足 ai​≤aj​≤bi​≤bj​。不难发现初始状态下的 (bi​−ai​)+(bj​−aj​) 就是最大贡献。

至此，此题结束，写完代码下一题去。

如果存在相交“区间”，就不停的选这两个“区间”，额外贡献为 0。

如果不存在，就找出相距最近的两个“区间”，额外贡献为两倍距离。其中的距离指的是靠后的最短点减去靠前的右端点。

代码如下：

#include<bits/stdc++.h>
using namespace std;
#define int long long
int t, n, m, ad; pair<int, int>v[200005]; bool hs;
signed main() {
	ios::sync_with_stdio(0);
	for (cin >> t; t; t--) {
		cin >> n >> m; //和 1 次没有区别
		for (int i = 1; i <= n; ++i) cin >> v[i].first;
		for (int i = 1; i <= n; ++i) cin >> v[i].second;
		for (int i = 1; i <= n; ++i)
		if (v[i].first > v[i].second) 
			swap(v[i].first, v[i].second);
		sort(v + 1, v + n + 1); ad = 3e9;
		for (int i = 1; i != n; ++i)
			ad = min(ad, max(v[i + 1].first - v[i].second, 0ll) << 1);
		for (int i = 1; i <= n; ++i) ad += v[i].second - v[i].first;
		cout << ad << endl;
	}
	return 0;
}
