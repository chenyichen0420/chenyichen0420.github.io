思路分析

考虑 DP，设 dpidp_idpi​ 表示凑出一个数字 iii 需要的数字数量。

由定义和范围限制，dpm+1=m+1dp_{m+1}=m+1dpm+1​=m+1 是显然的，随后写出递推公式：dpi=m+1+∑j=m+1i−1dpjdp_i=m+1+\sum_{j=m+1}^{i-1} dp_jdpi​=m+1+∑j=m+1i−1​dpj​。

于是类似这篇题解，我们直接使用数学归纳法，可以很轻松的证明出 dpi=2i−m−1×(m+1)dp_i=2^{i-m-1}\times(m+1)dpi​=2i−m−1×(m+1)。

于是套一个快速幂就做完了。注意特殊处理 n≤mn\le mn≤m。代码如下：

#include<iostream>
#include<vector>
#include<queue>
#include<cstring>
#include<string>
#include<algorithm>
using namespace std;
#define int long long
constexpr int mod = 998244353;
int t, n, m;
inline int qpow(int a, int b) {
	int ret = 1;
	for (;b;b >>= 1, a = a * a % mod)
		(b & 1) && (ret = ret * a % mod);
	return ret;
}
signed main() {
	ios::sync_with_stdio(0);
	for (cin >> t;t;t--) {
		cin >> n >> m;
		if (m >= n) cout << "1\n";
		else
			cout << (m + 1) * qpow(2, n - m - 1) % mod << endl;
	}
}
