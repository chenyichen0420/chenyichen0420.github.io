思路分析

一个符合直觉的事实是如果我有一个跨度非常大的询问区间，那么我大概率能够“感觉到”有一个弦，也就是最短路必须经过弦。

但是光这么说是没有用的，你得把他精准的找出来，而且步数不能太多。

先来估计一个粗略的上界吧。按照前面的直觉，显然我们需要一个长度为 $\frac{n}{2}$ 的查询区间。然后看看我们能干什么：

在 $n$ 较大的时候，我们将其近似的看作一个圆，于是只有弦的中垂线是不满足要求的，其他的任意一个直径都是满足要求的。

这也就启示我们，大概率区分不出来的长区间只有少数。是不是我随便找一个初始位置，然后随手转两下就能找到呢？

在 $2\mid n$ 的情况下这确实是成立的，但是在 $2\nmid n$ 的情况下却存在一个特殊情况：长度为 $2$ 的弦。这时候，在半数长度为 $\lfloor\frac{n}{2}\rfloor$ 的查询看来都是完全没有问题的。我们希望的就是在奇数情况下，这个弦可以向上拱一下，再向下拱一下，这样就能覆盖所有情况了。

这确实是不困难的，结合前面所说，我们可以严谨的说明最多只需要 $5$ 次操作（这是一个宽松的上界），我们就可以找到一个最段路必须经过弦的弦。

光知道经过最短路可是没有用的，我们还需要定位到最短路。直接旋转过去是不现实的，我们考虑能不能先把一个端点对上，再对另一个端点？

此时我们观察数据范围，容易注意到次数上限大约是 $\log$ 的，于是我们考虑能不能先将其中的一个端点给二分出来。

考虑之前找到的特殊端点为 $l,r$，我们确认 $l-1,r$ 是否能够使得距离变小，如果是，那么说明有一个端点在比 $l$ 更小一些的某个位置（这里是循环的，总之就是不停的缩小，在缩到 $r$ 之前就可以遇到端点）。反之则说明在比 $l$ 更大一些的某一个位置。

如何判断有没有跳过呢？如果 $l,r$ 的实际距离为 $d$，那么在正确方向上跳过之前，$l+\Delta d,r$ 的距离应该是 $d-\Delta d$。也就是说确实可以二分出端点。这部分需要的操作次数是 $\log n+1$ 的。

我们最后考虑另一个端点在哪里。如果当前 $l,r$ 的距离是 $d$，那么真正的 $r$ 就应该是 $r\pm(d-1)$ 中的某一个。判定就是二者真实距离为 $1$ 且二者环上距离不为 $1$，这部分的操作次数是 $1$。

于是总的操作次数就只有 $\log n+7=37$ 次，完全可以通过。代码如下：

CPP 
#include<bits/stdc++.h>
#include<cassert>
using namespace std;
#define int long long
int t,n, ret, cnt;
inline int ensure(int v){
	return (v+n*5-1)%n+1;
}
inline int dis(int l,int r){
	l=ensure(l); r=ensure(r);
	if(l>r) swap(l,r);
	return min(r-l,l+n-r);
}
inline int ask(int l,int r){
	if(dis(l,r)==1) return 1;
	l=ensure(l); r=ensure(r);
	cout<<"? "<<l<<" "<<r<<endl;
	int v; cin>>v; return v;
}
signed main(){
    ios::sync_with_stdio(0);
	for(cin>>t;t;t--){
		cin>>n; int p=1,da,p2=1+n/2,f=1; 
		if(n<=8){
			bool hs=0;
			for(int i=1;i<=n;++i)
				for(int j=i+1;j<=n&&!hs;++j)
					if(dis(i,j)!=1&&ask(i,j)==1)
						cout<<"! "<<i<<" "<<j<<endl, hs=1;
			cin>>ret;
			continue;
		}
		while(dis(p,p2)==(da=ask(p,p2))){
			if(n&1)
				if(f) p2++,f^=1;
				else p++,f^=1;
			else p++,p2++;
		}
		int l=0,r=n/2,mid;
		if(ask(p+1,p2)==da-1){
			while(l!=r){
				mid=l+r+1>>1;
				if(ask(p+mid,p2)==da-mid) l=mid;
				else r=mid-1;
			}
			p+=l;
		}
		else if(ask(p-1,p2)==da-1){
			while(l!=r){
				mid=l+r+1>>1;
				if(ask(p-mid,p2)==da-mid) l=mid;
				else r=mid-1;
			}
			p-=l;
		}
		p=ensure(p);
		da=ask(p,p2);
		if(dis(p,p2-da+1)!=1&&ask(p,p2-1)==da-1) p2-=da-1;
		else p2+=da-1;
		p2=ensure(p2); cout<<"! "<<p<<" "<<p2<<endl;
		cin>>ret;
	}
}
