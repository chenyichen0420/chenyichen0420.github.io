思路分析

任何一个经过点 (x,y) 的抛物线 y=x2+ax+b 都满足 y=x2+ax+b（废话）。

二次方程不是很好处理，所以有两种思路：

第一种：模拟凸壳暴力处理。

我们像正常的二维凸壳一样，维护末尾的抛物线，新加入点时如果发现在抛物线上就不停地删除抛物线直到满足要求。

第二种：转化凸壳正常处理。

y=x2+ax+b⟺y−x2=ab+b。因此我们将 (x,y) 重新标记为 (x,y−x2)，就变成了最多有多少条直线经过至少两个点而且上面没有别的点。

这就是问向上凸的凸壳的边数。正常求解即可。

代码如下：

#include<bits/stdc++.h>
using namespace std;
struct vecor {
	double x, y; vecor(double xi = 0, double yi = 0) :x(xi), y(yi) {}
	friend istream& operator>>(istream& l, vecor& r) { return l >> r.x >> r.y; }
	inline vecor operator-(const vecor& r) const { return vecor(x - r.x, y - r.y); }
	inline bool operator<(const vecor& r) { return x != r.x ? x < r.x : y < r.y; }
}p[100005];
int n, s[100005], sc, m;
inline double slope(const vecor& v) { return v.y / v.x; }
signed main() {
	ios::sync_with_stdio(0); cin >> n;
	for (int i = 1; i <= n; ++i) cin >> p[i], p[i].y -= p[i].x * p[i].x;
	sort(p + 1, p + n + 1); p[n + 1].x = 11451419;
	for (int i = 1; i <= n; ++i) if (p[i].x != p[i + 1].x) p[++m] = p[i];
	n = m; s[++sc] = 1;
	for (int i = 2; i <= m; ++i) {
		while (sc > 1 && slope(p[s[sc]] - p[s[sc - 1]]) <= slope(p[i] - p[s[sc - 1]])) sc--;
		s[++sc] = i;
	}
	cout << sc - 1 << endl;
}
