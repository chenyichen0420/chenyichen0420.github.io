题目大意

给你 nnn 个二元组 (xi,yi)(x_i,y_i)(xi​,yi​)，问你有几对二元组满足其中存在两个相同的元素。

思路分析

我们统计这些数中每个数出现了多少次。如果 xi=yix_i=y_ixi​=yi​ 就只统计一次（只能匹配一对）。那么，一个数出现了 ttt 次就意味着有 t×(t−1)2\frac{t\times (t-1)}{2}2t×(t−1)​ 对满足要求。

————吗？

有一种特殊情况，两个二元组完全一样（认为一个二元组是无序的），例如 (1,2)(1,2)(1,2)，(1,2)(1,2)(1,2)，(2,1)(2,1)(2,1) 三者两两相同。这种情况就会被多统计。那么，一个二元组出现了 ttt 次就意味着多统计了 t×(t−1)2\frac{t\times (t-1)}{2}2t×(t−1)​ 次，减掉就是了。

————吗？

还有细节，如果一个二元组满足 xi=yix_i=y_ixi​=yi​，它不会被重复统计。求重复组时不考虑就行了。

代码如下：

CPP 
#include<bits/stdc++.h>
using namespace std;
int n,a,b;long long ans; unordered_map<int,int>csv; unordered_map<long long,int>cfv;
int main(){
	ios::sync_with_stdio(0);
	cin>>n;
	while(n--){
		cin>>a>>b;
		if(a>b)a^=b^=a^=b;
		csv[a]++; csv[b]+=(a!=b);
		cfv[a*1000000000ll+b]+=(a!=b);
	}
	for(unordered_map<int,int>::iterator it=csv.begin();it!=csv.end();++it)
		ans+=(it->second)*1ll*((it->second)-1)>>1;
	for(unordered_map<long long,int>::iterator it=cfv.begin();it!=cfv.end();++it)
		ans-=(it->second)*1ll*((it->second)-1)>>1;
	cout<<ans<<endl;
	return 0;
}
