题目大意

给你一个长度为 nnn 的序列 aaa，可以进行任意多次操作，每一次可以选择一个区间，其众数个数严格大于区间长度的一半，并将这个区间所有数改为为这个众数。请问最终这个序列所有的数都一样的时候这个数可能是多少？无法都一样则输出 −1-1−1。多测。

思路分析

怎么说都知道搜索是错的，2×1052\times10^52×105 能给你放过去？！

这时候我们分析一下每一次操作：其众数个数严格大于区间长度的一半。

首先我们知道，如果两个一样的数放在一起，那么这个数一定是可以的。他可以选两个这个数，然后从相邻的地方强制选一个数加进来，变成这个数，以此类推：aabaabaab 变为 aaaaaaaaa。

我们就能发现，如果选的序列是偶数长度的，众数个数又在长度的一半以上，那么一定有相邻的。

如果是奇数长度呢？为了没有相邻的，这个串必须是这样：abacadaeaf…abacadaeaf\dotsabacadaeaf…。

我们随机抓取一段，比如 abaabaaba。我们发现其还是可以变成 aaaaaaaaa。

就此我们知道，只用列举长度为 333 的序列，找到一个出现两次的数就是一个可行的数。

代码如下：

CPP 
#include<bits/stdc++.h>
using namespace std;
int t, n, a[200005]; 
map<int, bool>ans; map<int, bool>::iterator it;
int main() {
	ios::sync_with_stdio(0);
	cin >> t;
	while (t--) {
		cin >> n; ans.clear();
		for (int i = 1; i <= n; ++i) {
			cin >> a[i];
			if (i > 1 && a[i] == a[i - 1]) ans[a[i]] = 1;
			if (i > 2 && a[i] == a[i - 2]) ans[a[i]] = 1;
		}
		for (it = ans.begin(); it != ans.end(); ++it)
			cout << it->first << " ";
		if (!ans.size()) cout << "-1";
		cout << endl;
	}
	return 0;
}
