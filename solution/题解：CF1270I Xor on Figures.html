思路分析

我们约定 ⨂/⊗ 表示异或运算，并重新定义二维卷积为异或二维卷积：Ai,j​=(B×C)i,j​=⨂x=02n−1​⨂y=02n−1​Bx,y​×Ci−xmod2n,j−ymod2n​。

这个东西有什么用呢？

我们令 Bxk​,yk​​=1，那么在 Ci,j​ 设置一个值 v 对 A 的影响就是：Axk​+i(mod2n),yk​+j(mod2n)​⊗Ci,j​→Axk​+i(mod2n),yk​+j(mod2n)​。

也就是说 C 的值实际上就对应着我们的一种合法的解。

直觉上本质不同的方案其实只有一个，严格证明较为困难，可以使用 WHT 严格证明上述运算的可逆性，在此略去。

总之，我们考虑利用 A 矩阵（也就是输入）和 B 矩阵（也就是那 m 个特殊点为 1 的矩阵）反推答案。

注意到 (B×B) 中为 1 的位置只有 (2×xk​,2×yk​)。因为其他的位置都可以通过对称性推出来是为 0。

那么显然 B2k=I。这里的 I 是卷积意义下的单位矩阵，即只有 I0,0​=1 的矩阵。那么得到 B 的逆矩阵为 B2k−1。

所以答案就是 A×B2k−1 中非 0 的位置个数。这里，我们使用快速幂的方法来分析，就会发现 B2k−1=∏i=0k−1​B2i，而前面的结论告诉我们 B2i 只有不超过 m 个 1。

因此时间复杂度可以做到 O(n×4n×m)。

所以出题人是怎么想到的呢？

代码如下：

#include<bits/stdc++.h>
using namespace std;
#define int long long
constexpr bool online=0;
int n,a[512][512],m,x[262145],y[262145],na[512][512],mod,ans;
signed main(){
    if(online)
        freopen("mc.in","r",stdin),
        freopen("mc.out","w",stdout);
    ios::sync_with_stdio(0); cin>>n;
    for(int i=0;i!=1ll<<n;++i)
        for(int j=0;j!=1ll<<n;++j)
            cin>>a[i][j];
    cin>>m; mod=(1ll<<n)-1;
    for(int i=1;i<=m;++i) cin>>x[i]>>y[i];
    for(int i=0;i!=n;++i){
        memset(na,0,sizeof na);
        for(int j=0;j!=1ll<<n;++j)
            for(int k=0;k!=1ll<<n;++k)
                for(int p=1;p<=m;++p)
                    na[j+x[p]&mod][k+y[p]&mod]^=a[j][k];
        memcpy(a,na,sizeof na);
        for(int p=1;p<=m;++p)
            x[p]=x[p]<<1&mod,
            y[p]=y[p]<<1&mod;
    }
    for(int i=0;i!=1ll<<n;++i)
        for(int j=0;j!=1ll<<n;++j)
            ans+=(bool)a[i][j];
    cout<<ans<<endl;
}
