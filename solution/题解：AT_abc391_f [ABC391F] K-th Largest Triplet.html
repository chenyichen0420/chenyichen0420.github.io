思路分析

要求的那个东西我简记为 fi,j,kf_{i,j,k}fi,j,k​，读过题的应该知道是啥。

将三个数组从大到小排序后，我们不难发现 fi,j,k≥fi+1,j,k,fi,j+1,k,fi,j,k+1f_{i,j,k}\ge f_{i+1,j,k},f_{i,j+1,k},f_{i,j,k+1}fi,j,k​≥fi+1,j,k​,fi,j+1,k​,fi,j,k+1​。用这个进行类推，不难发现 i′>i,j′>j,k′>ki^\prime>i,j^\prime>j,k^\prime>ki′>i,j′>j,k′>k 时恒有 fi,j,k≥fi′,j′,k′f_{i,j,k}\ge f_{i^\prime,j^\prime,k^\prime}fi,j,k​≥fi′,j′,k′​。

因此，i×j×k>mi\times j\times k>mi×j×k>m 的时候（这里的 mmm 是原题中的 KKK，防重名），所有的 fi′,j′,k′(i′≤i,j′≤j,k′≤k,不同时满足)≤fi,j,kf_{i^\prime,j^\prime,k^\prime}(i^\prime\le i,j^\prime\le j,k^\prime\le k,不同时满足)\le f_{i,j,k}fi′,j′,k′​(i′≤i,j′≤j,k′≤k,不同时满足)≤fi,j,k​。而显然，fi′,j′,k′f_{i^\prime,j^\prime,k^\prime}fi′,j′,k′​ 的个数 ≥m\ge m≥m。

所以我们只需要把满足 i×j×k≤mi\times j\times k\le mi×j×k≤m 的所有值存下来，再调用一遍 nth_element 就行了。

实践出真知，极限数据下这样的 i,j,ki,j,ki,j,k 只有 167827301678273016782730 组，而 nth_element 是线性的。三秒时限吸口氧随便跑。实测 666ms666ms666ms。

#define int long long
int n,m,a[200005],b[200005],c[200005];vector<int>v;
inline int calc(int x,int y,int z){
	return a[x]*b[y]+b[y]*c[z]+c[z]*a[x];
}
signed main(){
	ios::sync_with_stdio(0);

	for(int i=1;i<=n;++i) cin>>a[i];
	for(int i=1;i<=n;++i) cin>>b[i];
	for(int i=1;i<=n;++i) cin>>c[i];
	sort(a+1,a+n+1,greater<int>());
	sort(b+1,b+n+1,greater<int>());
	sort(c+1,c+n+1,greater<int>());
	for(int i=1;i<=m&&i<=n;++i)
