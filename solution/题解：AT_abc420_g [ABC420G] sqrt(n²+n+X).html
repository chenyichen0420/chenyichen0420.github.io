思路分析

神秘比赛，G 题比 F 题简单得多。

设 $k=\sqrt{n^2+n+x}$，其中 $k$ 是一个整数。那么 $n^2+n+x=k^2$，整理得 $n^2+n+x-k^2=0$。

我们可以将其视为一个二次方程，判别式为 $\Delta = 1-4(x-k^2)=1-4x+4k^2$。

为了使 $n$ 为整数，判别式 $\Delta$ 必须是一个完全平方数，且 $n$ 必须是整数。

因此，我们需要 $1-4x+4k^2=m^2$，其中 $m$ 是一个整数。整理得 $(2k-m)(2k+m)=4x-1$。

由于 $4x-1$ 是一个固定的整数，我们可以将 $4x-1$ 分解为两个因子的乘积，这两个因子的差为 $2m$，和为 $4k$。

设 $a=2k-m,b=2k+m$，则 $a\times b=4x-1,b-a=2m$，并且 $k=\frac{a+b}{4},m=\frac{b-a}{2}$。

由于 $k$ 和 $m$ 必须是整数，因此 $a+b$ 能被 4 整除，$b-a$ 能被 2 整除。

因此，我们可以枚举 $4x-1$ 的所有因子对 $(a,b)$，使得 $a\times b=4x-1$，并且 $a+b$ 能被 4 整除。

对于每个这样的因子对，我们可以计算 $m=\frac{b - a}{2}$，然后，我们可以求解 $n=\frac{-1\pm m}{2}$。

由于 $n$ 必须是整数，因此 $-1 \pm m$ 必须是偶数，即 $m$ 必须是奇数。因此，我们需要 $m$ 是奇数。

算法流程：

  1. 计算 $D = 4x - 1$。
  2. 枚举所有包含负数的满足 $a\times b=D$ 的整数对 $(a,b)$。
  3. 对于每个因子对 $(a,b)$：
    • 检查 $a+b$ 是否能被 4 整除。
    • 计算 $k=(a+b)/4,m=(b-a)/2$。
    • 检查 $m$ 是否为奇数。
    • 计算 $n=(-1\pm m)/2$ 并收集。
  4. 去重排序后输出。

代码如下：

CPP 
#include<bits/stdc++.h>
using namespace std;
#define int long long
int n, d, k, m, tn; vector<int>v;
signed main() {
	ios::sync_with_stdio(0);
	cin >> n; tn = abs(n = 4 * n - 1);
	for (int i = 1; i * i <= tn; ++i)
		if (tn % i == 0) {
			for (int k = 1; k <= 1; ++k) {
				int a = i, b = n / i;
				if ((a + b) % 4) break;
				int m = (b - a) / 2;
				if (m % 2 == 0) break;
				v.emplace_back((-1 + m) / 2);
				v.emplace_back((-1 - m) / 2);
			}
			for (int k = 1; k <= 1; ++k) {
				int a = -i, b = -n / i;
				if ((a + b) % 4) break;
				int m = (b - a) / 2;
				if (m % 2 == 0) break;
				v.emplace_back((-1 + m) / 2);
				v.emplace_back((-1 - m) / 2);
			}
		}
	sort(v.begin(), v.end());
	v.erase(unique(v.begin(), v.end()), v.end());
	cout << v.size() << endl;
	for (int i : v) cout << i << ' ';
}
