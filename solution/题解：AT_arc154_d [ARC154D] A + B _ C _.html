思路分析

直接做肯定是不好做的，考虑一下有没有什么办法让排序更简单。

显然有，如果 ai​>aj​，那么一定有 ai​+1>aj​，也就是说，假如说 ap​=1,ai​>aj​，那么输出 ? i p k 一定返回 Yes。可以证明若 ai​<aj​，则返回 No。

因此，我们只需要找出 1 的位置就行了。而且，令 ap​=1，我们显然有 ap​+ap​≤ai​(i=p),ai​+ai​>ap​(i=p)。

所以说我们假设其位置为 p=1，然后向后扫一遍，如果查询 p p i 返回 Yes，则将 p 更新为 i。由上面的两个引理显然可得这个位置就是 1。

思路理清了。代码如下：

#include<bits/stdc++.h>
using namespace std;
char c[5];
inline bool que(const int&l,const int&r,const int&v){
	cout<<"? "<<l<<" "<<r<<" "<<v<<endl;
	cin>>c; return c[0]=='Y';
}
int n,mp,a[2005],at[2005];
int main(){
	cin>>n; mp=1;
	for(int i=2;i<=n;++i)
		if(que(mp,mp,i)) mp=i;
	for(int i=1;i<=n;++i) a[i]=i;
	stable_sort(a+1,a+n+1,[](const int&l,const int&r){
		return que(r,mp,l);
	});cout<<"! ";
	for(int i=1;i<=n;++i) at[a[i]]=i;
	for(int i=1;i<=n;++i) cout<<at[i]<<" ";
	fflush(stdout);
}
